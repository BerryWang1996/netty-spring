# RC4a MeshBroker (TCP transport skeleton) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or
> superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `MeshBroker` — a `ClusterBroker` implemented over direct Netty TCP (drop-in for `RedisPubSubBroker`;
Redis registry/heartbeat stay off the message hot path) — proven by a two-node real-TCP E2E.

**Architecture:** One Netty TCP server per node (inbound = receive) + a lazily-cached outbound connection per peer
(send), length-prefix framed, carrying the existing HMAC-wrapped `EnvelopeCodec` line. Peer addresses come from a new
Redis-backed `MeshNodeDirectory`; membership is derived (live-by-heartbeat ∩ has-address). RC4a broadcast is naive
(all peers); interest-routing is RC4b.

**Tech Stack:** Spring Boot 2.7.18, Netty 4.1 (core dep — `LengthFieldBasedFrameDecoder`/`LengthFieldPrepender`,
`WriteBufferWaterMark` built-in, **zero new deps**), Lettuce 6.1, JDK 17, JUnit 5, Mockito, real Redis on
localhost:16379.

**Spec:** `docs/superpowers/specs/2026-06-15-mesh-rc4a-design.md` (design-review-folded). **Branch:**
`feature/1.10.0-mesh-rc4a` (spec + review archive already committed at `8ad7a3e`).

prevent OOM from a garbage length prefix (M2).

//    same string out.
// 2. partialRead_reassembles: split the framed bytes into two ByteBuf chunks → decoder yields one full frame.

- [ ] **Step 3: Implement `MeshFrames`:**
```java
package com.github.berrywang1996.netty.spring.web.websocket.cluster.mesh;
// imports: io.netty.handler.codec.{LengthFieldBasedFrameDecoder,LengthFieldPrepender}, ChannelPipeline, etc.
public final class MeshFrames {
    private MeshFrames() {}
    /** Inbound: 4-byte length header, strip it; cap at maxFrameBytes (reject larger → TooLongFrameException). */
    public static LengthFieldBasedFrameDecoder decoder(int maxFrameBytes) {
        return new LengthFieldBasedFrameDecoder(maxFrameBytes, 0, 4, 0, 4);
    }
    /** Outbound: prepend a 4-byte length header. */
    public static LengthFieldPrepender prepender() { return new LengthFieldPrepender(4); }
    /** Encodes a wrapped codec line to a length-framed ByteBuf payload (UTF-8). The prepender adds the length. */
    public static io.netty.buffer.ByteBuf toPayload(io.netty.channel.ChannelHandlerContext ctx, String wrapped) {
        return io.netty.buffer.Unpooled.copiedBuffer(wrapped, java.nio.charset.StandardCharsets.UTF_8);
    }
    public static String fromPayload(io.netty.buffer.ByteBuf frame) {
        return frame.toString(java.nio.charset.StandardCharsets.UTF_8);
    }
}
```

- [ ] **Step 4: Run → PASS.**
- [ ] **Step 5: Commit** `feat(cluster/mesh): length-prefix framing + inbound frame cap (RC4a M2 T1)`.

  - `subscribe(uri, l)` / `subscribeUnicast(nodeId, l)`: put in `Map<String, ClusterMessageListener> broadcastListeners`

    `unsubscribe()`.

    for now dispatch inline by kind: `UNICAST/CLOSE/ROOM_BROADCAST` → unicastListener; `BROADCAST/PRESENCE_CHANGE` →
    broadcastListeners.get(env.getUri()) (PRESENCE_CHANGE rides PRESENCE_CHANNEL as its uri). Null listener → drop.
  - `state()`: `ACTIVE` while bound (Task 6 adds DEGRADED); `shutdown()`: close server + outbound channels + groups +
    directory.remove(nodeId) + state→SHUTDOWN.

## Task 6: Mesh total-isolation degrade trigger (M5)

**Files:** Modify `.../cluster/mesh/MeshBroker.java`; Test `.../cluster/mesh/MeshDegradeTest.java`.

The mesh fires `onTransportLost()` / `state()==DEGRADED` **only on total isolation** (directory has peers it should
reach, but zero are reachable), so the existing `on-redis-loss`/grace/`DEGRADED` machinery has a meaningful trigger; a
single dead peer is per-target (M1 drop + `onPublishFailure`), not a global degrade.

- [ ] **Step 1: Failing test** `MeshDegradeTest`:
```java
// 1. singleDeadPeer_staysActive: 2 peers in directory, 1 outbound connect fails → state()==ACTIVE, listener NOT
//    notified onTransportLost.
// 2. totalIsolation_degradesAndNotifies: all peer connects fail (directory non-empty) → state()==DEGRADED,
//    transportStateListener.onTransportLost() fired once.
// 3. recovery_restores: a peer becomes reachable again → state()==ACTIVE, onTransportRestored() fired.
// 4. noPeers_staysActive: empty directory (single-node cluster) → ACTIVE, no degrade (isolation needs peers-it-should-reach).
```

- [ ] **Step 2: Run → FAIL.**

- [ ] **Step 3: Implement.** Track per-peer reachability (last connect/write success). On the refresh tick (or on
  connect-failure), compute `shouldReach = directory.peers(self).size()`; `reachable = count of peers with a live
  outbound channel`. `if (shouldReach > 0 && reachable == 0)` → CAS state ACTIVE→DEGRADED + `listener.onTransportLost()`
  once. `if (state==DEGRADED && reachable > 0)` → CAS DEGRADED→ACTIVE + `onTransportRestored()`. Wire
  `setTransportStateListener` to store the listener (mirrors `RedisPubSubBroker`).

- [ ] **Step 4: Run → PASS.**
- [ ] **Step 5: Commit** `feat(cluster/mesh): total-isolation degrade trigger — onTransportLost only when zero of N peers reachable (RC4a M5 T6)`.
